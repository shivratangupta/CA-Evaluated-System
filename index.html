CA Evaluated System

About

This Programme is written in C++ Programming Language. It was part of a Mini-Project at Lovely Professional University, India. And it was selected as the best CA Evaluated System within other 30 Mini-Projects.
This programme takes all student's academic data (i.e. Continuous Assesment scores, Mid-term scores, Projects score) and evaluates their Weightage marks according to University's rule using a simple algorithm.
